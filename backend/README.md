# Flask Project - Backend

## Adminer

To visualize the SQLite database, open terminal and go to the [./static], run the command:

```console
$ php -S localhost:8080
```

Then you can access adminer: (http://localhost:8080/adminer.php)

In the login page, select SQLite for database type, and enter the SQLite database file path (e.g [../static/mydatabase.db]).
